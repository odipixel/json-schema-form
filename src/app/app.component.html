JSON Schema Form

JSON Schema form

It can do two things.

  • Create a form directly from a JSON schema.
  • Validate form fields against that same JSON schema.

How to use ?

  • Change the schema in 'schema' input field.Changes will automatically reflect in 'generated form' box
  • Fill the input with wrong value to check the validations.Submit button will be active, only if all the form fileds are valid.
  • Fill the form with valid inputs and click on submit button.
